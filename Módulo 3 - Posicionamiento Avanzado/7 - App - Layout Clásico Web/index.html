Posicionamiento Avanzado

En esta práctica verás cómo funcionan static, relative, absolute, fixed y sticky en ejemplos reales.

Static (comportamiento por defecto)

Los elementos con position: static siguen el flujo normal del documento (este texto + elementos se apilan).

Relative (crea un contexto)

Este contenedor tiene position: relative. Dentro colocamos un elemento absolute (badge) que se posiciona respecto a este contenedor.

ABS

Mové el tamaño de la ventana o cambiá el texto para ver cómo la posición del badge depende del contenedor relativo.

Absolute (fuera del flujo)

Un elemento con position: absolute se saca del flujo y se posiciona respecto a su ancestro posicionado. Si no existe, se posiciona respecto al viewport.

Flota (absolute)

Fixed (pegado al viewport)

Header y botón flotante usan position: fixed — permanecen visibles al hacer scroll.

Esto es útil para elementos de acción persistente (CTA, controles globales).

Sticky (mezcla relative/fixed)

El sidebar está usando position: sticky en desktop: se queda visible cuando se alcanza su posición y luego vuelve al flujo.

Si querés ver sticky en acción con más scroll, hacé scroll largo abajo.
